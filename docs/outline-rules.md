# Outline Rules

## Keyword Rules

- The main keyword is the keyword with the highest volume in the cluster.
- The main keyword must appear in the title.
- Secondary keywords must have lower volume than the main keyword.
- Secondary keywords should appear naturally in H2/H3-level outline items when possible.
- Do not force awkward secondary keywords into headings.
- Exclude keywords with volume `0`; keep keywords with volume `10`.

## Outline Rules

- Use numbered headings, not `H2` / `H3` labels.
- Use `Title`, then `1.`, `1.1`, `1.2`, `2.`, `2.1`, etc.
- Keep outlines useful but not too dense.
- For most blog posts, use 7-10 main sections.
- Include FAQ only when search intent or competitors justify it.
- Match blog intent. Do not turn blog keywords into service/category page outlines.

## New Site Strategy

- Prefer lower KD and narrower topics for new sites.
- Avoid broad head terms unless they are used as supporting keywords.
- Pick topics where the site can add expertise, examples, or operational insight.
